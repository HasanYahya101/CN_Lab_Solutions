ls – Lists files and directories in the current directory.
cd – Changes the current directory. Example: cd Documents moves into the "Documents" folder.
pwd – Prints the current working directory (shows the full path of where you are).
mkdir – Creates a new directory (folder). Example: mkdir new_folder creates a directory named "new_folder".
rmdir – Removes an empty directory. Example: rmdir old_folder deletes "old_folder" if it's empty.
rm – Removes files or directories. Example: rm file.txt deletes "file.txt", and rm -r folder_name deletes a directory and its contents.
cp – Copies files or directories. Example: cp file1.txt file2.txt makes a copy of "file1.txt" named "file2.txt".
mv – Moves or renames files/directories. Example: mv file.txt new_location/ moves "file.txt" to "new_location".
touch – Creates an empty file or updates the timestamp of an existing file. Example: touch newfile.txt creates "newfile.txt" if it doesn’t exist.
